# Azure FinOps Agent — "Painful Job" Slide Image Prompts

Square 1:1 (1024×1024) cartoon images for the **"FinOps today is a painful job"** slide of the Azure FinOps Agent pitch deck. Each image visualizes one specific pain bucket, anchored in Azure-cloud imagery (cloud servers, spend, portals, dashboards) — not generic office props. Generate in order; feed each finished image as the visual reference for the next so the character "Sam" stays consistent.

Pitch context: Sam is an Azure FinOps practitioner (cost owner / cloud admin / FinOps lead). The narrative shows him drowning in today's manual reality — which is exactly what the Azure FinOps Agent solves.
